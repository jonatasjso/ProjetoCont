# Dashboard PHP

Dashboard web para visualização de dados e indicadores, focado no controle e acompanhamento de contratados, desenvolvido com PHP no backend e tecnologias web no frontend.

## Tecnologias Utilizadas
- PHP
- HTML5
- CSS3
- JavaScript
- Chart.js
- Fonte Poppins

##  Estrutura do Projeto
/
├── css/        # Estilos da aplicação
├── js/         # Scripts JavaScript
├── font/       # Fontes (Poppins)
├── img/        # Imagens
├── pages/      # Páginas internas
├── plugins/    # Bibliotecas externas
└── index.php   # Página principal

## Como Executar o Projeto
1. Instale um servidor local (XAMPP, WAMP ou Laragon)
2. Coloque a pasta do projeto dentro de `htdocs`
3. Acesse pelo navegador:
http://localhost/nome-do-projeto
